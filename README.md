# SLAI Project

## Overview
SLAI is an open-source project initializing its development phase. The goal is to build a debuggable, modular, and cleanly separated architecture.

## Version
Current version: **0.0.1**
*(Version bumps to 1.0.0 will only be made when authorized).*

## Project Tenets
- **Open Source First**: We prioritize exclusively open-source sub-dependencies and libraries whenever possible.
- **Data Privacy**: No private information (keys, generic credentials, internal URIs) should ever be committed. We use `.env` files for configuration.
- **Human & AI Readable**: Our docs and code are clean, structured, and heavily commented/documented in Markdown.

## Quick Start
1. Clone the repository
2. See `.env.example` to set up your environment variables
3. Check the `/docs` folder for architectural insights and roadmap planning.
